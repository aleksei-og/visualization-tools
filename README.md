# visualization-tools
# Отчет по продажам в Power BI

Данный проект представляет собой визуальный аналитический дашборд, созданный в Power BI на основе данных о продажах товаров международного магазина. Проект включает обработку исходных данных, построение модели данных, создание вычисляемых мер и разработку визуализаций для анализа ключевых бизнес-показателей.

## Источник данных

В качестве основы использован датасет Global Super Store:  
https://www.kaggle.com/datasets/apoorvaappz/global-super-store-dataset

В проект были загружены следующие таблицы:

- Orders (основная таблица заказов)
- Shipping (данные об отправке товаров)
- Order Details (детализация заказов)
- Regions (география продаж)
- DateTable (календарная таблица)
- Products (товарный каталог)
- Customers (информация о клиентах)
- CSV-таблица в качестве примера импорта данных из другого формата

## Преобразование данных

Перед созданием визуализаций были выполнены следующие действия:

- Корректировка типов данных и локалей для числовых значений
- Добавление вычисляемого столбца Год заказа на основе даты
- Извлечение текстовых значений для формирования столбца «Бренд»
- Создание столбца «Категория приоритета» с помощью выражений DAX
- Дублирование столбцов для упрощения построения связей
- Формирование структуры модели данных по схеме «Снежинка»

## DAX-меры

Созданы ключевые показатели для анализа:

- Total Sales — общий объем продаж
- Total Profit — общая прибыль
- Average Discount — средняя скидка по заказу
- High Priority Sales — продажи высокоприоритетных заказов
- Profit Margin — маржинальность (%)
- Top Product Name — самый прибыльный продукт

При создании мер использовались функции:
SUM, AVERAGE, CALCULATE, IF, YEAR, RELATED, DIVIDE и др.

## Визуализация данных

Отчет включает несколько вкладок с различными визуальными элементами:

### Plots
- Столбчатая диаграмма продаж по категориям
- Круговая диаграмма распределения заказов по регионам
- Линейный график прибыльности во времени
- Таблица детализации

### Statistic Cards
- Набор карточек с ключевыми KPI (продажи, прибыль, скидка, маржа, топ-продукт)

## Модель данных

Использована схема **«Снежинка»**, где таблица фактов связана с таблицами-справочниками.  
Такой подход упрощает масштабирование и улучшает аналитическую гибкость отчетов.

## Итог

В результате:

- Построена полноценная модель данных Power BI
- Выполнены преобразования данных и логика расчета метрик
- Реализован аналитический дашборд для изучения продаж, маржинальности и поведения клиентов
- Проект позволяет оперативно анализировать результаты бизнеса по регионам, категориям и продуктовым направлениям
